# 实时转录 LiveTranscriber

macOS 原生应用：现场对话实时转录成 Markdown，中英双向翻译，英文译文可语音播报——专为跨语言面对面交流设计的「口译 + 速记」工具。

**全程本地运行，数据不出本机**：语音识别（Whisper / 苹果引擎）、中英互译、语音合成三个环节全部跑在 Mac 上，不调用任何云端 API，不上传任何录音，零订阅零费用。

## 功能特性

- 🎙️ **实时转录**：麦克风收音，说话停顿后自动成段，带时间戳；说话过程中显示灰色实时草稿
- 🧠 **双识别引擎**：
  - **Whisper（默认）**：OpenAI Whisper large-v3-turbo 本地模型，逐句自动判断中/英文，混说识别准确率高
  - **苹果（快速）**：macOS 26 系统识别，秒级出字延迟最低，需手动指定语言
- 🌐 **自动翻译**：英文发言自动附中文译文（系统离线翻译模型）
- 🗣️ **对话翻译模式**：你说中文 → 自动翻成英文 → **扬声器语音播报给对方**；播报期间自动暂停收音防回声
- 💬 **聊天式界面**：对话模式下对方（英文）靠左蓝色气泡、我（中文）靠右绿色气泡，一眼分清谁说的
- 📝 **Markdown 存档**：停止后自动保存双语对照记录到 `文稿/实时转录/`，方便复盘
- 🎨 **个性化**：深色/浅色模式、字体（苹方/宋体/楷体/圆体/等宽）、字号、文字颜色可调，深色模式下自动保护文字可读性
- 🔊 **英文音色可选**：调用系统语音，支持试听，可下载接近真人的高级音色

## 系统要求

- macOS 26 (Tahoe) 或更新版本（依赖新版 SpeechAnalyzer 接口）
- Apple 芯片或 Intel 芯片均可（Whisper 在 Apple 芯片上快得多）
- 首次使用 Whisper 需联网下载约 1.6GB 模型（仅一次）

## 安装

### 从源码构建（需要 Xcode 命令行工具）

```bash
git clone https://github.com/frankchn-x/live-transcriber.git
cd live-transcriber
./build-app.sh
cp -R dist/实时转录.app /Applications/
open /Applications/实时转录.app
```

### 安装别人分享给你的 .app

1. 把 `实时转录.app` 拖进「应用程序」文件夹
2. 首次打开若提示「无法验证开发者」：**系统设置 › 隐私与安全性**，拉到底部点「仍要打开」
3. 若提示「应用已损坏」：终端执行下面命令后重新打开

```bash
xattr -cr /Applications/实时转录.app
```

## 首次使用

1. 点击底部圆形麦克风按钮，系统会请求**麦克风权限**，点允许
2. Whisper 引擎首次使用会弹窗确认**下载识别模型**（约 1.6GB，状态栏有进度条；官方源失败自动切国内镜像）。不想下载可临时改用苹果引擎
3. 模型下载后首次加载需要几分钟编译优化（仅一次），之后秒级就绪
4. 首次翻译会提示下载系统**离线翻译模型**（很小），同意即可

## 使用说明

### 场景一：跟说英文的朋友面对面聊天（核心场景）

1. 点右上角**滑杆按钮**打开设置：识别引擎选 **Whisper（中英自动）**，打开**对话翻译**开关，选好**播报音色**（点 ▶ 可试听）
2. 电脑放两人中间，音量开够，点麦克风按钮开始
3. **朋友说英文** → 左侧蓝色气泡显示英文原文 + 中文译文（你看屏幕）
4. **你说中文** → 右侧绿色气泡显示中文 + 英文译文，**同时扬声器自动把英文念给朋友听**
5. 注意：播报英文的几秒钟里麦克风暂停收音，请朋友等播完再接话
6. 结束点停止按钮，完整双语记录自动保存

### 场景二：纯记录（会议、讲座）

关闭「对话翻译」开关即可：只转文字不播报，英文内容仍自动附中文译文，段落为全宽卡片样式。

### Markdown 输出

停止后自动保存到 `~/Documents/实时转录/转录_日期_时间.md`，也可随时点右下角复制按钮。格式：

```markdown
# 现场转录 2026-06-12 18:30

- **[18:30:05]** Hey, how have you been recently?
  > 译：嘿，你最近怎么样？
- **[18:30:12]** 我最近在做一个翻译软件。
  > EN: I've been working on a translation app recently.
```

晚到的译文会自动补写回文件，记录保证完整。

### 设置面板（右上角滑杆按钮）

| 分组 | 选项 | 说明 |
|---|---|---|
| 转录 | 识别引擎 | Whisper（准、中英自动）/ 苹果（快、需选语言），转录中锁定 |
| | 对话翻译 | 开 = 口译模式（中文翻英文并播报）；关 = 纯记录 |
| | 播报音色 | 仅对话翻译开启时显示，▶ 试听 |
| 外观与文字 | 外观 | 跟随系统 / 浅色 / 深色 |
| | 字体/字号/颜色 | 实时生效，带预览；看不清的自定义颜色自动回退默认 |

### 下载更高质量的英文音色

**系统设置 › 辅助功能 › 朗读内容 › 系统声音 ⓘ › English › Voice**，推荐下载 **Ava (Premium)**、**Zoe (Premium)**、**Evan (Enhanced)**、**Nathan (Enhanced)**（认准括号里的 Premium/Enhanced，▶ 可先试听）。下载后重开应用即出现在音色列表。注意 Siri 声音不开放给第三方应用，下了也用不了。

## 常见问题

**Q: Whisper 模型下载失败？**
应用会自动切换国内镜像（hf-mirror.com）重试。仍失败请检查网络，或临时改用苹果引擎。

**Q: 说完话要等 2~3 秒才出文字？**
Whisper 是整句识别（停顿 1.2 秒后定稿），这是精度与延迟的取舍。要最低延迟切苹果引擎。

**Q: 中文为主夹杂英文单词的句子为什么不翻译？**
按设计只有整段英文（如对方发言）才触发翻译，中文为主的混说句子你自己能看懂，不加噪音。

**Q: 播报时对方说的话没录上？**
是防回声机制（否则喇叭播的英文会被识别再翻译，无限循环）。请等播报结束再说话。

**Q: 重新构建后麦克风权限又弹一次？**
ad-hoc 签名每次构建会变化，系统会重新确认，点允许即可。

## 技术架构

```
麦克风(AVAudioEngine) → 格式转换(16kHz) → 识别引擎
  ├─ Whisper: 能量VAD切句 → 语言检测(zh/en钳制) → WhisperKit/CoreML 整句识别
  └─ 苹果: SpeechAnalyzer 流式识别
→ 语言判断 → Translation框架翻译(en↔zh) → AVSpeechSynthesizer播报(防回声闸门)
→ SwiftUI 界面 + Markdown 存档
```

- 纯 SwiftUI + Swift Package Manager 构建，无需完整 Xcode
- 唯一第三方依赖：[WhisperKit](https://github.com/argmaxinc/WhisperKit)（Whisper 的 CoreML 实现）
- 关键经验：Whisper 解码管线内置语言检测不可靠（失败回退英文会把中文意译掉），需每句先调独立检测接口并钳制到 zh/en 二选一

```bash

```

## 隐私

所有处理（识别、翻译、合成）均在本机完成，应用不含任何网络上报代码；联网仅发生在首次下载模型时（HuggingFace / Apple 服务器）。转录内容只保存在你的 `文稿/实时转录/` 目录。
